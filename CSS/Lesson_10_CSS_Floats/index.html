When creating animations and other GPU-heavy actions, it's important to understand the willchange attribute. Both CSS keyframes and the transition property use GPU acceleration.

Performance is increased by offloading calculations to the device's GPU. This is done by creating paint layers (parts of the page that are individually rendered) that are offloaded to the GPU to be calculated. The willchange property tells the browser what will animate, allowing the browser to create smaller paint areas, thus increasing performance.
